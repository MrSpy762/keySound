This app is designed as a free version of Soundpad!

1. Download the keySound archive
2. Unzip the archive to a convenient location
3. Run keySound.exe
4. Select "Add Sound" at the top and select the desired file, then name it
5. Add a hotkey if needed

To add or remove a hotkey for a sound, right-click the desired sound and select either "Assign a global hotkey" or "Remove a global hotkey"
